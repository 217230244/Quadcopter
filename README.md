#  小型四轴无人机控制系统

基于 STM32 的四轴无人机飞控项目，实现姿态解算、串级PID控制、2.4G无线通信与FreeRTOS多任务调度。

---

##  项目简介

本项目基于 STM32 单片机开发了一款成本低、易量产的小型四轴无人机控制系统：
- 采用 MPU6050 六轴传感器采集运动数据，通过滤波算法与四元数解算实现姿态估计；
- 使用串级 PID 算法实现无人机姿态稳定控制，通过定时器 PWM 输出控制有刷电机；
- 搭载 2.4G 无线通信模块实现遥控与飞控的数据交互，配合 FreeRTOS 实现多任务调度；
- 集成 OLED 显示、电压采样、失控保护等功能，完成完整的无人机飞行控制流程。

##  核心功能实现

### 1. 片上外设驱动开发
- 基于 STM32CubeMX 配置并实现了 **IIC、SPI、ADC、定时器PWM、UART** 等外设驱动；
- 完成定时器 PWM 输出配置，实现 4 路电机的转速控制；
- 配置 ADC 采集电池电压，实现电压采样与平滑滤波。

### 2. 传感器数据处理与姿态解算
- 实现 MPU6050 的初始化、零偏校准，读取三轴加速度与角速度原始数据；
- 实现一阶低通滤波、滑动窗口滤波、卡尔曼滤波，降低传感器噪声；
- 通过四元数算法解算欧拉角，完成姿态融合，为PID控制提供可靠数据。

### 3. 串级PID姿态控制
- 实现串级PID算法，外环控制角度、内环控制角速度，实现无人机姿态稳定；
- 对PID参数进行调试优化，实现无人机的悬停、姿态保持与平稳飞行；
- 支持按键微调PID参数，方便现场调试与性能优化。

### 4. 2.4G无线通信与遥控控制
- 实现 Si24R1 无线模块驱动，完成飞控与遥控器之间的数据收发；
- 自定义通信协议，实现摇杆数据的采集、滤波、换算、中点校准与限幅；
- 实现通信失联安全处理逻辑，防止无人机失控坠落。

### 5. FreeRTOS 多任务调度
- 完成 FreeRTOS 系统移植与功能裁剪，实现多任务调度；
- 划分传感器采集、姿态解算、PID控制、无线通信、OLED显示等独立任务；
- 实现任务间同步与数据传输，保证系统实时性与稳定性。

### 6. 附加功能实现
- 12864 OLED 显示驱动，实现姿态数据、电压状态、系统信息的实时显示；
- 灯控系统实现飞行状态指示；
- 解锁指令设计，防止误操作导致的意外启动。

##  快速上手

### 硬件准备
- STM32 开发板（如 STM32F103 系列）
- MPU6050 六轴传感器模块
- Si24R1/NRF24L01 2.4G 无线模块 ×2（飞控端+遥控端）
- 12864 OLED 显示屏
- 有刷电机 ×4、电调/电机驱动板、机架、电池等无人机配件

### 编译与烧录
1.  使用 STM32CubeMX 生成基础工程，配置对应外设与 FreeRTOS；
2.  将本项目代码导入 Keil MDK，配置工程路径与编译选项；
3.  编译生成 `.hex` 文件，通过 ST-Link 烧录到开发板；
4.  按架构连接硬件，先调试传感器与通信模块，再进行电机控制与PID参数调试。
